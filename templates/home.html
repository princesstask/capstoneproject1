Simulator & Media Pembelajaran Teori Bahasa dan Automata

Platform interaktif berbasis Web Flask untuk memodelkan, menyimulasikan, dan menganalisis konsep mesin otomata serta tata bahasa formal secara visual dan praktis.

4 Modul Pembelajaran Wajib

Silakan pilih salah satu modul di bawah ini untuk mulai belajar dan menguji string pada simulator.

Finite State Automata

Eksplorasi mesin pengenal string otomatis tanpa memori. Dilengkapi dengan 4 tab simulator interaktif: DFA, NFA, Moore, dan Mealy.

Buka Simulator FSA

Regular Expression

Pelajari pencocokan pola string (Regex) beserta generatator otomatis aturan produksi Grammar Reguler Kanan (Right-Linear Grammar) yang setara.

Buka Modul Regex

Pushdown Automata

Simulasikan mekanisme otomata dengan memori tambahan berupa Stack (operasi Push/Pop) untuk mengenali Tata Bahasa Bebas Konteks (CFG).

Buka Modul PDA

Chomsky Normal Form

Media pembelajaran algoritma transformasi CFG sembarang menjadi Bentuk Normal Chomsky (CNF) langkah demi langkah secara terstruktur.

Buka Modul CNF

Tentang Proyek Ini

Aplikasi web ini dirancang sebagai pemenuhan tugas akhir berbasis Outcome-Based Education (OBE). Tujuan utamanya adalah menjembatani teori matematika formal yang abstrak dalam perkuliahan Teori Bahasa dan Automata agar lebih mudah dipahami melalui visualisasi komputasi langsung di peramban web.
